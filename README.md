# -UIDAI-Data-Biometric-Analysis
UIDAI Aadhaar Biometric Analysis is a data analytics project using SQL and Power BI to study biometric enrolment trends across India. It analyzes age distribution, regional load, seasonal patterns, and operational efficiency, helping optimize infrastructure, improve coverage, and support data-driven governance.

🚀 Project Overview
This project combines:

🧮 SQL-based data processing & analytics
📊 Power BI dashboard visualization
🧠 Insight generation for planning and optimization
The aim is to help UIDAI understand enrolment patterns, identify high-load regions, and improve biometric registration efficiency.

🗂 Dataset & Data Processing
Multiple CSV files merged using SQL
Data cleaning performed:
Standardized state names
Removed inconsistencies
Structured into biometric_data table :contentReference[oaicite:0]{index=0}
📌 Key Columns:
enroll_date
states, districts, pincode
age_5_17, age_17_plus
📊 Dashboard Features
🔹 KPI Metrics

🔹 Visual Analysis
📈 Enrolment trend over time
🍩 Age group distribution
📊 Monthly and quarterly growth
🗺 State & district-level mapping
🧠 Key Insights
📊 Enrolment distribution is balanced between 5–17 and 17+ age groups
📅 Higher enrolments in Q2 and Q4 indicate seasonal demand
🏙 States like Uttar Pradesh, Maharashtra, Bihar dominate enrolments
⚙️ Regional load imbalance highlights infrastructure needs
📉 Some districts show lower performance, indicating coverage gaps
